# U.S. Presidential Election Data Visualization (1976-2020)

This project visualizes U.S. Presidential Election data from 1976 to 2020, highlighting popular vote percentages and Electoral College outcomes.

## Features

- Line chart showing popular vote shares by party
- Bar chart for Electoral College results
- Ready-to-use CSV data

## Setup

```bash
git clone https://github.com/adit019/us-presidential-election-viz.git
cd us-presidential-election-viz
pip install -r requirements.txt
python visualize_election.py
```

## Data Source

- [Kaggle: US Presidential Elections 1976-2020](https://www.kaggle.com/datasets)
- [MIT Election Lab](https://electionlab.mit.edu/)
